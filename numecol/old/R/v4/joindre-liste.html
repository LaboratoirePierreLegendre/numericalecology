Mailing Lists

I maintain two mailing lists, one for beta-testers and one for release notifications. The main difference is that I send bug reports to the beta list, as well as preliminary info. Both lists get release notes. If you would like to join either list, please send a blank email message to:
  • r-announce-l-on@biol09.biol.umontreal.ca to join the "Announce" list, or
  • r-beta-l-on@biol09.biol.umontreal.ca to join the "Beta-testers" list
That's it! You should receive a confirmation message in your mailbox shortly thereafter. If not, please use my feedback form or send a message to casgrain@invite.umontreal.ca
