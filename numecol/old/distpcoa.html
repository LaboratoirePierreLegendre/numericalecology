Note: The R Package, version 4.0 now does Principal Coordinates Analysis with the two negative eigenvalue correction methods implemented in this program.

Documents available for distribution include:

On-line version of the documentation

Macintosh version
Self-extracting binhexed archive containing:
  • Fortran source code
  • Compiled versions of the program for Macintosh computers using 680x0, 680x0 + FPU, or PowerPC processor
  • Program documentation, in Word 6 format
Note The 680x0 versions can handle 250 objects, the PowerPC version 400 objects. Should that be insufficient for your needs, you are invited to modify the source code and re-compile it.
32-bit DOS version
Zip-compressed archive containing:
  • Fortran source code
  • Compiled version of the program for 32-bit computers (Windows 95 or NT)
  • Program documentation, in Word 6 format
