R 4.0 for Windows

There is a Windows version of The R Package currently in development. This version, intended for Windows 95 and later version of the operating system, will have a graphical user interface built with Borland's Delphi. The computational core of R will simply be compiled into a library (DLL) which will be required by the user interface. This core will be shared between the MacOS and the Windows version, which will ensure a high quality of computational accuracy by leveraging the 15+ years of development of R on the MacOS, VAX and VM/CMS. Furthermore, having only one source base will simplify bug management when they are found. To be informed as soon as the first (development) version of the R Package for Windows is released, you can subscribe to the mailing list by sending a blank email message to:
r-windows-l-on@biol09.biol.umontreal.ca
Or you can check back on this website. In the meantime, please see our emulation page.
