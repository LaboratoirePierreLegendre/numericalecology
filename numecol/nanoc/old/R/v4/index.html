The R Package for multivariate and spatial analysis

The R Package is a free program for the MacOS that will allow you to perform complex multivariate and spatial statistical analyses with ease. For instance, with the R Package you can:

  • Compute similarity, distance and correlation (R-mode) matrices
  • Compute geographic distances from latitude-longitude data
  • Create a principal coordinates analysis (PCA)
  • Create a correspondance analysis (CA)
  • Compute a Mantel test
  • Create a hierarchical or K-Means cluster analysis
  • Study spatial autocorrelation processes
  • Verify and normalize your data matrices
  • And much more! R 4.0 is a major overhaul of R 3.01. A Windows version is currently in development; in the meantime, you can visit our emulation page From here, you can:
