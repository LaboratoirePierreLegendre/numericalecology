Description of the Oribatid mite data set

Download the Oribatid mite data set This data set contains 3 files. Data on a line are separated by tabs. 1) Oribatida: 35 species (rows) x 70 sites (columns). 2) Coord.xy: 70 sites (rows) x 2 coordinates, in m, in Cartesian plane (columns). 3) Environ: 70 sites (rows) x 5 environmental variables (columns):
Substrate (nominal variable):
  1. Sphagnum group 1
  2. Sphagnum group 2
  3. Sphagnum group 3
  4. Sphagnum group 4
  5. Litter
  6. Bare peat
  7. Interfaces between any of the previous
Shrubs (nominal variable):
  1. None
  2. A few
  3. Many
Microtopography of the substratum (binary variable):
  1. Blanket (flat)
  2. Hummock (raised)
Substratum density in g.L-1 of dry uncompressed matter (quantitative variable) Water content of the substratum, in g.L-1 (quantitative variable)
Please refer to the following papers if you use or refer to this data set in a thesis or a scientific publication:
  • Borcard, D., P. Legendre and P. Drapeau. 1992. Partialling out the spatial component of ecological variation. Ecology 73: 1045-1055.

  • Borcard, D. and P. Legendre. 1994. Environmental control and spatial structure in ecological communities: an example using Oribatid mites (Acari, Oribatei). Environmental and Ecological Statistics 1: 37-53.
