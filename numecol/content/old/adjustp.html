Adjusted P-values

Pierre Legendre
January 2002
Département de sciences biologiques
Université de Montréal
This program computes various forms of adjustments to a list of probability values in cases of multiple testing. The theory was reviewed by Wright (1992) and by Legendre & Legendre (1998). A page from Legendre & Legendre (1998) explaining how the calculations are done is provided in appendix to this user's guide. Documents available for distribution include:
  • Macintosh version
    • Fortran source code
    • Compiled versions of the program for Macintosh computers using 680x0 or PowerPC processor
    • Program documentation, Adobe Acrobat format
    • Sample files

  • DOS/Text-only version
    • Fortran source code.
    • 32-bit compiled version of the program for DOS computers
    • Program documentation, Adobe Acrobat format
    • Sample files
