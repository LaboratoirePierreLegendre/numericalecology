The R Package Version 3 for VAX computers

Everything necessary for a successful compilation and use of the package is available below. The files are compressed with zip 2.01, and should be transferred as binaries. If you do not have the unzip 5.0 decompression utility, you may download it from: ftp://ftp.uu.net/pub/archiving/zip/VMS/ Pascal sources are bilingual. Simply select the .com files in the language of your choice.

Notes

  1. The current programmer of R version 4 is totally clueless in how this VAX version works. For VAX-specific questions, please ask Pierre Legendre.
  2. The documentation is available in Macintosh self-reading, (see the Macintosh version), Word for Windows 1 and PostScript (ps) formats.
  3. European users: the documentation is setup as US letter. Unfortunately, an A4 version is not available.
