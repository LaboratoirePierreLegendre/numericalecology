Old Statistical Programs Download Page

Note: The date indicates the last time the page was modified.
System requirements are as follows:
  • Macintosh programs require System 7 (Mac Plus and up) and have been tested under MacOS 8.1-9.1 (Classic under MacOS X). R 3.0, however, runs best under System 6.
  • All DOS programs require a 32-bit environment, available under Windows 95/98/NT. Unless noted, they are all "console" application, and do not generate graphical output, only ASCII text.
  • Linux users, please download the DOS version and compile the source code with your Fortran compiler (f77 or g77). You will need to unzip the archive and remove the carriage returns from the source code.
